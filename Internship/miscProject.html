Remind Me

RemindMe is a timer mod to send a chat message to the player in the specified time to the player By typing in the command "/Remind", they can then set a timer for 1,5,15,30 or 60 minutes later. Adding additional comments after "/Remind" will also be given to the timer, so that the player can remember why they set the timer in the first place.

Player Info

This is a simple mod that displays the player's coordinates within the game world, the direction they are facing, and their clicks per second. The compass in the mod faces the same directions as Minecraft, except that it uses the interCardinal rather than just the normal cardinal points in the game.
